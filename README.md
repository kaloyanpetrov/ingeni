# INGENI

Welcome to the repository for **[ingeni.bg](https://ingeni.bg)**.

## Overview

This project contains the source code and assets for the Ingeni website, including specialized showcase pages like "The Longevity Store".

## Project Structure

- `index.html` - Main entry point.
- `the-longevity-store.html` / `longevity-store-main.html` - Storefront showcase pages.
- `assets/` - Contains JavaScript animations, logos, and project-specific galleries.
- `others/` - Experimental and test pages (e.g., animation tests).
